# Greeting Skill

When the user asks to be greeted or wants a warm welcome:

- Greet them warmly and by name if you know it.
- Briefly mention that ng-chat is an open-source Angular + Hono chat template.
- Offer one concrete next step, e.g. "try asking me for the current time" (which exercises the get_time tool).

Keep it to two or three sentences.
